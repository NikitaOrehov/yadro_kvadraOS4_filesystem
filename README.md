# Scaner

## Описание
Приложение на языке С++ под Linux, которое через равные интервалы времени будет обходить
домашний каталог и находить в нем мультимедийные файлы (изображения, аудио, видео) и
формировать файл в домашнем каталоге .media_files в формате json следующего вида: 
{ "audio": [ "111.mp3", "222.wav" ], "video": [ "333.mpg" ] "images": [ "444.jpeg", "555.
png" ] }.
Также предусмотрена опция, когда json-файл не создается на файловой системе, а доступен
 через HTTP по адресу http://localhost:1234/media_files через GET-запрос.

## Требования
- Компилятор с поддержкой C++17: **GCC 9+** или **Clang 7+**
- **CMake 3.14** или выше
- Операционная система: **Linux** (включая WSL)

## Сборка
- Склонируйте репозиторий - https://github.com/NikitaOrehov/yadro_kvadraOS4_filesystem.git 
- Перейдите в папку куда склонировали
- Соберите программу
    - mkdir build && cd build
    - cmake ..
    - make

## Запуск
Теперь в папке build находится исполняемый файл scanner

**Режим по умолчнию**

Введите ./scanner для запуска режима по умолчанию. 
В этом режиме программа сканирует $HOME каждые 30 секунд и записывает результат в
~/.media_files

**Параметры запуска**

--http   Запускает HTTP сервер вместо записи в файл.

--skip-hidden   Пропускать скрытые каталоги (имя начинается с точки). По умолчанию не
 задан (скрытые обходятся).

--interval <N> Интервал между сканированиями в секундах (по умолчанию 30).

--path <путь> Каталог для сканирования (по умолчанию $HOME).

## Остановка
Для остановки работы программы необходимо нажать cntr + c


## Пример
./scanner --http --path ~/Videos --interval 60

Программа запускается в HTTP режиме и сканирует каталог ~/Videos с интервалом в 60 секунд.

Для получения резуьтата выполните curl http://localhost:1234/media_files
или перейдите по адресу http://localhost:1234/media_files.
